# Harry Potter - Java RPG System

This project is a basic simulation of characters and spells from the Harry Potter universe, developed in Java using Object-Oriented Programming (OOP) principles while following the Strategy Pattern.

---

## Project Structure

The program is organized into the following packages and classes:

### 1. Caracteres
This package contains the entities representing the wizards available for user selection.
* **HarryPotter**: Class representing the character Harry.
* **RonWeasley**: Class representing the character Ron.
* **HermioneGranger**: Class representing the character Hermione.

### 2. Spells
This package groups the logic for the various spells that the player can equip and cast.
* **ExpectoPatronum**: Implementation of the Patronus Charm.
* **Expelliarmus**: Implementation of the Disarming Charm.
* **OculusReparo**: Implementation of the Repairing Charm.
* **WingardiumLeviosa**: Implementation of the Levitation Charm.

### 3. Test
This package contains the main entry point of the system.
* **Main**: Includes the executable method where the interactive menu, game state management, and console input logic reside.

---

## Execution
To start the program, ensure all classes are compiled and run the **Main** class located within the **test** package.
